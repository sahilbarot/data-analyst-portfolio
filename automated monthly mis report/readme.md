# Project 3 — Financial MIS Automation

## What This Does
Automatically generates a professionally formatted
Excel MIS report from raw monthly financial data.

## Business Problem Solved
Finance teams spend hours every month building the
same report manually. This script does it in seconds.

## Tools
Python · pandas · openpyxl · Excel

## Key Metrics Calculated
- Net Profit and Profit Margin %
- Month-on-Month % change
- Year-to-date cumulative totals
- Revenue per headcount
- Best and worst performing months

## Business Relevance
Same workflow used by MIS analysts in Ahmedabad IT
companies, GIFT City operations teams, and financial
reporting departments globally.
